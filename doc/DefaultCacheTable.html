Class: DefaultCacheTable

DefaultCacheTable(conf)

Class: DefaultCacheTable

-- default cache table behavior -- abstract method framework

This is a default (stopgap) caching object, provided to act as the cache. But, a more robust verision of this object may be provided via configuration with the understanding that the provided object will implement the same interface as the default object.

Constructor

new DefaultCacheTable(conf)

DefaultCacheTable.constructor

Initializes global maps, one for files and one for directories. All files and directories are found by path (or hash of path) in these global maps.

When directories are maintained, sub directories and files within the maintained directory will be accessed by the directory object. The file will be found in the file cache and the directories will be found in the directory cache.
