DefaultCacheTable(conf)

Class: DefaultCacheTable

-- default cache table behavior -- abstract method framework

This is a default (stopgap) caching object, provided to act as the cache. But, a more robust verision this object may be provided via configuration with the understanding that the provided object will implement the same interface as the default object.
