FileOperationsCache

FileOperationsCache

class: FileOperationsCache extends FileOperations

The FileOperationsCache class operates with a cache table class of a known interface. Here is the list of methods that CacheTable classes will need to implement in order to replace the default cache table.

Provides a class with override capability that specializes methods in FileOperations to provide caching with possible reading/writing having to do with cache hits.

When the FileOperationsCache class is used by the application, the methods of FileOperations remain in affect, but they will perform caching operations. The methods listed below are either the overrides of above methods or they are added in relation to keeping objects in memory.

The methods, overriden, will be called by methods listed above. Example, output_json will make a call to write_out_json.
